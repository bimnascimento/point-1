Installation, Activation & Update

Steps For Installation And Activation

You can install plugin either of these two methods.

New install via FTP

  1. Download the plugin from CodeCanyon, and extract it. You should find a directory arforms.
  2. In your WordPress installation, upload arforms to wp-content/plugins/
  3. Activate the ARForms plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin from CodeCanyon
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on choose file and browse the download package from CodeCanyon. When the plugin uploads successfully, click on Activate Plugin.

After activating the plugin, you will be able to see ARForms menu in admin left menu.

After activating the plugin, you need to activate your license for support and enables automatic upgrades. A license key only be used for one installation of WordPress at a time.

Click on General Settings link to activate your license. You will see below section there on that page. To activate your license, you need to enter your Name, Email address and purchase code( Unique code obtained while purchased this product).

Steps for Manual Update

  1. Download the plugin from CodeCanyon, and extract it. You should find a directory arforms.
  2. In your WordPress installation, replace existing arforms directory with latest downloaded arforms directory to wp-content/plugins/ folder.
  3. After replacing folder, Click on ARForms Menu from WordPress admin panel and plugin will be updated with latest changes.

Conditional Logic

Conditional Logic is a way to show, hide, enable, disable form fields by applying rules on it which depends on the user's input. If you want to show or hide any particular field (or group of fields (sections) ) based on the rule you have applied, then conditional logic is used to achieve this functionality. You can set conditional logic on any fields except Hidden field, Captcha and Image.

To add conditional logic on any field, click on Field Options icon which you will get by hover on any field. By clicking on Field Options icon, you will be able to see 3 tabs which are: 'Basic Options, 'CSS Property' and 'Conditional Law'. Click on 'Conditional Law' tab, and you can set rules for that field from there.

For Example, You have created contact form. And you want to show "Email" field only if user's preferred contact method is Email. To achieve this, Click on Field Options icon of Email Field, and add rule as displayed in below image.

Show/Hide ( Action )

There is dropdown for actions to be applied on field, select one of them as per your requirement.

  • Show - By selecting this option, target field will be displayed only if specific condition is fulfilled.
  • Hide - By selecting this option, target field will be hidden only if specified condition is fulfilled.

All / Any: ( Condition)

The condition is part of the overall conditional logic, in addition to the actual rules, that determine what action to take.

  • All - When you have applied more than one condition on target field. And you want that each condition should be fulfilled, at that time this option is used.
  • Any - When you have applied more than one condition on target field. And you want that any of them condition should be fulfilled, at that time this option is used.

Rules:

  • Field -Here you have provided list of fields that you have taken in your form. Some fields will not included like HTML, Page break, Captcha, Range Slider.
  • Operator -This is the operation which determines whether a value is evaluated towards your condition. Options available are: "equals", "not equals", "greater than", "less than", "contains", "not contains"
  • Value -This is an Option from the selected Field Name which the user must select or not select, based on the Operator.

Math Logic / Running Total

ARForms provides facility of Math Logic / Running Total that will perform run time calculation based on user's input.

To add math logic, you just need to add HTML control to editor. You can add formula for calculation by clicking on checkbox Enable Running Total.

Enable Running Total:

When you want to add formulas for math logic, you have to check Enable Running Total checkbox. When you check this checkbox, it will automatically add a pair of <arftotal></arftotal> tag. ( Opening and closing tag. ) where your cursor is pointing.

After checked the Enable Running Total, Please confirm that your cursor is pointing between the <arftotal> tag. You need to add formula between opening and closing tag.

After enabling the running total, you will be able to see a sample formula as displayed in below picture: Also, you will get Form fields to add in formula along with math operators for calculation.

Add Field:

This is the list of all fields which you have already added in editor for your form. Select any field which you want to use for calculation. By selecting field, it will be added to your formula which you can see in HTML content area. Here some inputs will not included such as Range slider, Page Break, Captcha.


Math Operators:

{ + } Sign:

It will perform addition operation between fields(operands). Just simply click on button and it will be added to your formula where your cursor is pointing.

{ - } Sign:

It will perform subtraction operation between fields(operands). Just simply click on button and it will be added to your formula where your cursor is pointing.

{ * } Sign:

It will perform multiplication operation between fields(operands). Just simply click on button and it will be added to your formula where your cursor is pointing.

{ / } Sign:

It will perform division operation between fields(operands). Just simply click on button and it will be added to your formula where your cursor is pointing.

{ ( ) } Brackets:

Starting and closing bracket used to differentiates and give high priority to particular operation between formulas.

example
([Product:123]*[Qty:125]) + 5

Validate Formula:

After adding the required fields and operators to the formula, you may want to check that this formula is valid or not. So, this button will check whether your formula is valid or not. If your formula contains any incorrect expression, then it will show you error message.

Form Settings

You can manage what action should be performed after form submission. You will get mainly 3 options for submission method.

You can display a message on page itself after form submission. That means, A message will be displayed on same page after submitting the form. This method will not redirect to any page.

Here you also have facility to set message at confirmation message option.

Page will be redirected to a entered URL after form submission. That means, External page will be displayed after form submission.

If you have used facility of sending your form data to custom url then here other facility is provided that is Show post value using shortcode given there.

Existing page content from your site will be displayed after form submission. That means, After submitting the form, If you want to just display thank you message from your existing page, you can use this option.

You can manage user email content and subject line and other settings from this area which will be sent after form is submitted from front end. 3) Email Marketers :

You can map Email Marketer(s) like Aweber, MailChimp, GetResponse, iContact, Constant Contact, GVO, EbizAC with the form from this area.

Also, you can set different settings than global setting for that particular form only. While creating new form, settings will be copied from Global Settings and than you can change settings according to your requirement for that particular form.

In API method, you can change your list name if you want to add subscriber into another list than global setting.

In Webform method, you can add different form value than global setting.

From this area, you can map form's field to Email Marketers. Map Email Marketers' first name, last name and email field with form fields.

4) Custom CSS:

From this area, you can give custom styling to form according to your need. You can add your own styling properties to form entities.

Note : Apply properties directly. Do not define class here.

  4. Give title of widget, select the form you wish to display and its relevant options in sidebar and click on "Save" button.

3) Insert Form into Theme

You can insert a form into page from your current theme / template file by calling the function. You will get all functions listed at ADD To Site tab at form editor.

For Normal View:

< ?php echo maincontroller::get_form_shortcode(array('id' => FORMID)); ?>

For Pop-up View:

< ?php echo 
maincontroller::get_form_shortcode_popup(array('id' => 'FORMID', 
'desc'=>'Click here to open Form', 'type'=>'link', 'height'=>'auto', 
'width'=>'800')); ?>

Note : Possible argument for type argument

  1. type="link"
  2. type="button"

For Sticky view:

< ?php echo 
maincontroller::get_form_shortcode_popup(array('id' => 'FORMID', 
'desc'=>'Click here to open Form', 'type'=>'sticky', 'position'=>'top', 
'height'=>'auto', 'width'=>'800', 'bgcolor'=>'#93979d', 
'txtcolor'=>'#ffffff')); ?>

Note : Possible argument for position argument

  1. position="top"
  2. position="bottom"
  3. position="left"
  4. position="right"

For Fly View:

< ?php echo 
maincontroller::get_form_shortcode_popup(array('id' => 'FORMID', 
'desc'=>'Click here to open Form', 'type'=>'fly', 'position'=>'left', 
'height'=>'auto', 'width'=>'800', 'bgcolor'=>'#2d6dae', 
'txtcolor'=>'#ffffff')); ?>

Note : Possible argument for position argument

  1. position="left"
  2. position="right"

FORMID will be replaced with your generated form id.

So, THAT'S IT! Your form is ready to publish. Happy Form Building!!
