## Description ##

 #### Dokan Menu Hider for **Dokan** plugin.
	
> You have to install [**Dokan Lite**](https://wordpress.org/plugins/dokan-lite/) plugin or you can get the [**Pro Version**](http://wedevs.com/theme/dokan/?utm_source=wporg&utm_medium=cta&utm_campaign=dokan-product-duplicator) to run this plugin

Manage seller dashboard menu or prduct tab 
* Disable seller dashboard menu
* Disable product edit tab for the seller

Author:  [@nayemDevs](http://nayemdevs.com)
Follow me on [@twitter](https://twitter.com/nayemDevs)

### Features:  ###

* Hide seller dashboard menu from admin panel
* Disable tab from the product edit page 
* Seller restriction 

== Installation ==

* Install Dokan plugin 
* Then Install Dokan Menu Hider 
* After installing, Please navigate to Wp admin-> Dokan->Settings->Selling Options. In this 
tab you will get both options- Hide Menu and Remove tab with a dropdown select.
